## C95 Decision Architecture

**Renewal motion (automatic if thresholds met):** Non-police resolution ≥75% + stable complaint volume (±10% variance) across two consecutive quarterly periods. No additional vote if framework conditions clear.

**Expansion amendment (separate vote if gates clear):** D2 inclusion contingent on DA escalation protocol signature + OPD training certification (Montez sign-off) + D2 community advisory charter + monthly public reporting mechanism. Expansion target C95, implementation C96 if vote passes.

**Cost of not deciding:** Pilot stalls, trust erodes, political coalition fractures (Mayor loses CRC coalition credibility if renewal-only strategy continues without expansion path).
