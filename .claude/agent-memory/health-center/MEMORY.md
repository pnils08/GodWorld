## Key Decision Logic (Cycle 86)
**Why these three moves now?**
1. **Architect RFP** — Design timeline is critical path. State's HCAI plan review window is 6-12 months. Must have design docs ready as soon as possible.
2. **Site due diligence parallel** — Don't wait for geotech to engage architects. Run both streams simultaneously. Geotech informs design, not the gate.
3. **Community meeting** — Legitimize the project as neighborhood-driven. Service priorities documented for operator coordination. Trust-building starts now.

**Momentum imperative**: 5-cycle delay pushed the entire schedule. No slack remains. Every month of delay costs $190K in soft costs and defers clinic opening by one month.
