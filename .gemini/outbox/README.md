# agy outbox — work pending the editor's review

agy drops finished depth pieces here for the primary editor (Mags) to review and ingest.

- One piece per file: `c{cycle}-<slug>.md` (in-world cycle, e.g. `c98-vela-profile.md`), opening with a 3-line header:
  `Subject:` · `Citizens (POP-IDs):` · `Beat:`
- **Presence in this folder = pending review.** agy does NOT ingest or publish.
- After the editor reviews and runs the pipeline, the piece moves to `ingested/`.

This is a handoff queue, not a doc catalog — the filesystem is the queue.
