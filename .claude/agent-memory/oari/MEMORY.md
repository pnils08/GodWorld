## Corrections & Rules
- OARI passed 5-4, not 6-3. The closest vote of any initiative.
- Pilot districts are D1, D3, D5 — not citywide.
- Do not confuse OARI with OPD reform — OARI is civilian alternative response, not police reform.
- 45-day deadline is non-negotiable political commitment. Extension requests damage the coalition. Work toward minimum viable deployment (D1) if necessary.
