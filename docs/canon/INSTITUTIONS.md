# Canon Institutions — Oakland-as-Simulated 2041

**Read at boot:** every generative agent and reviewer agent. Companion to [[canon/CANON_RULES]].

This file is the tier-organized roster of structural institutions. When agents need to name a hospital, school district, courthouse, construction firm, union, regulatory body, or large agency, they query this file. Rows tell you which tier the entity is in and (for tier 2) what canon-substitute applies.

---

## Conventions

Each domain has up to three sections:

- **Tier 1 — Use Real Names.** Public-geographic functions. Use directly, no substitute needed. Names listed are canon-permissible.
- **Tier 2 — Canon-Substitute Required.** Branded private entities. Each row maps Real Name → Canon Substitute → Status (`canon` / `proposed` / `TBD`). When `TBD`, escalate per CANON_RULES §Escalation rather than fabricate.
- **Tier 3** is handled by rule (no real individuals, ever) and isn't listed here.

When filling a tier-2 canon substitute, pick something that fits the prosperity-era frame. The name should sound like it could have existed in 2041 Oakland-as-simulated.
