# docs/entities/ — Agent-owned entity wiki pages

Individual citizen, council-member, business, or faith-org pages when Supermemory (world-data or bay-tribune) isn't the right home. Use this folder for entity records that need human editing, long narrative form, or deliberate canon preservation outside MCP retrieval.

Most citizen data lives in `world-data` Supermemory + `Simulation_Ledger`. Only write here when a page needs to be a durable, human-maintained doc.

Three-layer role: **agent-owned synthesis** (Phase 41.3).
